A personal Discord chat bot with multiple LLM backends, written in Rust. 

```shell
# Latest Release
docker pull ghcr.io/m1sk9/ferrisphone:latest

# Minor Release
docker pull ghcr.io/m1sk9/ferrisphone:v0

# Specific Release
docker pull ghcr.io/m1sk9/ferrisphone:v0.1.0
```

[_API Support: requires Discord API v10_](https://discord.com/developers/docs/reference#api-versioning)
